# Unzip backend #

Zip processor

## Supported Zip features ##
Supported methods:
- Store
- Shrink
- Reduce
- Implode
- Deflate
- Enhanced deflate (Deflate64)
- Bzip2
- LZMA (LZMA1)
- Zstandard
- XZ (LZMA2)

Supported encryption types:
- Traditional PKWARE encryption
- WinZip AE-1 and AE-2 (AES128, AES192, AES256)

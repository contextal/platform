# Image processing backend #
Backend processing popular image formats

# Supported features #
- Reading image format
- Reading image dimensions
- Reading image pixel format
- Reading exif metadata (TIFF, JPEG, PNG, WebP)
- Optional OCR

# Supported file types #
- BMP
- EXR (OpenEXR)
- GIF
- HDR
- ICO
- JPEG
- PNG
- PNM
- TIFF
- WebP
